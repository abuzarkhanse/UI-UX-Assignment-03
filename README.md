# UI/UX Assignment 3 - GitHub UX Audit

## Overview
This repository is created for Assignment 3 of the UI/UX Design course. The purpose of this assignment is to perform a UX structural audit of GitHub based on real user tasks.

## Objective
The goal is to analyze GitHub's:

- Navigation Design
- Interaction Design (IxD)
